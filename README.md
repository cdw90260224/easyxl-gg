# EasyXL.GG - 당신의 엑셀, 일상언어로 완벽하게 제어하세요

**EasyXL.GG**는 복잡한 엑셀 작업을 AI와 자연어 대화만으로 해결할 수 있게 돕는 차세대 데이터 분석 대시보드입니다.

## 🚀 주요 기능
- **자연어 검색 (Natural Language Search)**: "서울 지역 매출 합계 구해줘"와 같은 일상 언어로 데이터를 즉시 필터링하고 분석합니다.
- **AI 스마트 인사이트**: 업로드된 데이터 구조를 자동으로 파악하여 시각화 차트와 실행 계획을 제안합니다.
- **Privacy-First (개인정보 보호)**: 모든 데이터는 서버에 저장되지 않고 브라우저 로컬 메모리에서만 처리되므로 안전합니다.
- **엑셀 내보내기**: 분석된 데이터를 다시 세련된 엑셀 파일로 내보낼 수 있습니다.

## 🛠 Tech Stack
- **Frontend**: React (Vite), TypeScript, Tailwind CSS
- **AI Engine**: OpenAI API (GPT-4o-mini)
- **Data Handling**: XLSX, Recharts
- **Icons/UI**: Lucide React, Sonner (Toasts)

## 📦 시작하기 (Quick Start)
1. 저장소를 클론합니다.
2. `.env.example` 파일을 복사하여 `.env` 파일을 만들고 OpenAI API Key를 입력합니다.
3. 의존성을 설치합니다: `npm install`
4. 로컬 서버를 실행합니다: `npm run dev`
